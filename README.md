# 数据分析 — VAN类车西北五省上险销量分析

VAN 类车在西北五省(陕西、内蒙古、甘肃、宁夏、青海)的上险销量数据分析。基于上险数据底表,按既定模板生成 2026 年 1-4 月销量分析报告(9 个 sheet)。

## 目录结构

| 文件 | 说明 |
|---|---|
| `core.py` | 数据加载与通用计算函数(销量/同比/占有率/占同比/比重) |
| `engine.py` | 列规格(colspec)驱动的指标计算引擎 |
| `wr.py` | xlsx 写入与样式工具(合并表头、分级、配色) |
| `build.py` | 9 个 sheet 的结构定义与构建主程序 |
| `26年1-4月分析结果.xlsx` | 最终分析结果 |

## 运行

```bash
python3 build.py
```

依赖:`pandas`、`openpyxl`。需在同级目录提供底表 `5月上险数据底表.xlsx`(脚本中 `core.py` 的 `RAW` 路径)。

## 分析口径

- **统计期间**:2026 年 1-4 月;同比对比 2025 年 1-4 月(sheet5/8 的"2025 全年"列单独取全年)
- **销量** = 底表 `数量` 字段求和;0 值留空不显示
- **同比** = 本期销量 ÷ 去年同期销量 − 1(百分比,1 位小数);去年同期为 0 且本期有量 → "净增长"
- **占有率** = 该品牌销量 ÷ 同列条件下所有品牌总销量
- **占同比** = 本期占有率 − 去年同期占有率
- **比重**(sheet4/6/9)= 本行销量 ÷ 上一级父分类汇总销量
- 品牌按 2026 年 1-4 月销量降序,保留全部有销量品牌

## 9 个 Sheet

| Sheet | 主题 | 左侧分级 |
|---|---|---|
| 1 | 企业品牌总览 | 企业品牌 |
| 2 | 分省 × 品牌 | 省份 → 企业品牌 |
| 3 | 分省 × 福田对比 | 省份 |
| 4 | 省 → 城市 → 燃料(含比重) | 省份 → 城市 → 燃料种类 |
| 5 | 大VAN 场景分析 | 燃料 → 米段 → 品牌(仅大VAN品牌) |
| 6 | 大VAN 米段 × 功能(含比重) | 米段 → 功能分类 |
| 7 | 中VAN 明盲窗分析 | 明盲窗 → 品牌(仅中VAN品牌) |
| 8 | 风景G系列(非纯电中VAN) | 燃料种类 → 品牌 |
| 9 | 10类专用(含比重) | 10类专用 |

## 底表使用字段

VAN种类、大中VAN、年份、月份、省份、城市、企业品牌、米段(轴距划分)、功能、功能分类2、10类专用、燃料种类/燃料种类2、数量、容积段(新)、窗
